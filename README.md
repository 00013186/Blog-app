# BLOG APPLICATION

This app allows user to create blog, update the notes and delete them if they do not need anymore.

#### Instructions 
1.Download the source code 
```bash 
git clone [https://github.com/00013186/Blog-app.git]
```
or just download *.zip file

2.Install deps
```bash
npm install
```

3.Run the app
```bash
node app
```

#### Deps list
- express
- pug
